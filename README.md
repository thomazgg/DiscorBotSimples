## **Instalação | Como usar o bot**

 **1.**  Instale [node.js v16](https://nodejs.org/en/) ou superior

 **2.** Baixe este repositório e descompacte-o | ou git clone
 
 **3.** Preencha tudo em **`config.json`**
 
 <br/>

### *Modificar - config.json*

```javascript
{
    "token": "Bot_Token",
    "prefix": "Bot_Prefix"
}        
```
